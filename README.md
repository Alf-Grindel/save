# social匹配系统 - Save



## 技术栈

1. golang  + gorilla + rediStore
2. gorm 
3. MySQL  
4. Redis + ResSync



## 功能

1. 用户管理
   1. 用户注册 / 登录 / 登出 / 更新 / 注销
   2. 推荐用户 （预加载缓存， 定时更新缓存）- 分布式锁
2. 标签管理
   1. 用户可以添加、修改标签
   2. 允许用户根据标签搜索其他用户
3. 组队
   1. 创建队伍
   2. 加入队伍
   3. 查询队伍
   4. 更新队伍
4. 允许用户去修改标签
5. 推荐
   1. 相似度计算算法 完成接口调用
   2. todo 推荐用户方法添加相似度算法
